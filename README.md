# pay-to-contract-cli

## Description

Pay to Contract is a contract management cli tool.

## Usage

To install pay-to-contract from npm, run:

```
$ git clone git@github.com:commerceblock/pay-to-contract.git
$ cd pay-to-contract
$ npm install
$ ./bin/ptc --help
```
